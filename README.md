# Ephemeridis

Ephemeridis est une application iOS d’éphéméride moderne, élégante et hors ligne.

Elle propose :
- le calendrier du jour
- les fêtes et dictons
- les heures de lever et coucher du soleil
- les phases de la lune
- des informations METAR (météo aviation)
- une utilisation principalement hors ligne
